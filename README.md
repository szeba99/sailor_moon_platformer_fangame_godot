# 🌙 Sailor Moon Platformer Fangame

Welcome to the official repository of the **Sailor Moon Platformer Fangame**, a non-profit, open-source fan project. This game is a tribute to the Sailor Moon universe, built with passion using the **Godot 4.6** engine.

In a twist on the classic formula, you play as **Sailor Moon** on a mission to save **Tuxedo Mask** — think of it as a reverse Mario game where the princess saves the prince!

---

## 🚀 Project Status: Under Development

The game is currently in an **early development phase**. Many assets are temporary placeholders and will be replaced as the project matures:

*   **Visuals:** Current character sprites are sourced from the fan-game *La Luna Splende*. I am working on original, high-resolution hand-drawn assets to replace the temporary pixel art.
*   **Music:** Background tracks are currently provided by *Animesynth*.
*   **Environments:** Some tilesets are placeholders from *OpenGameArt.com*.

---

## ✨ Features

*   **Classic Platforming:** Authentic retro-style action.
*   **Cross-Platform:** Playable on Desktop and Mobile browsers with full touch support.
*   **Highly Moddable:** The project structure is designed to be easily modified. Assets are kept in unzipped folders for quick access.
*   **Level Editor:** Create your own levels using Godot's built-in TileMap system.

---

## 🛠️ How to Create Your Own Levels

Making a custom map is extremely easy:

1.  **Clone/Download** this repository.
2.  Open the project in **Godot 4.6** (or use the included `Editor.bat` and put the Godot Editor exe file next to it, I couldn't include it here because it's 180mb).
3.  Create a new scene in the `scenes/` folder.
4.  Drag and drop the `starter_pack.tscn` file (from `prefabs/basic/`) into your scene.
5.  **Start Building:** Use the TileMap editors to draw the terrain, place enemies from the scene collection, or drag-and-drop prefabs directly into your level.
